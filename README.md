# SaaSImperialBarber - Sistema de Gestion Premium

SaaS para barberias que buscan optimizar su tiempo y recursos: citas, clientes, inventario y comisiones.

## Sitio en vivo

**https://imperial-barber-tau.vercel.app**

> Credenciales de demo: `admin@imperialbarber.com` / `Admin123!`

## Stack

| Capa | Tecnologia |
|------|------------|
| Frontend | React + Vite + Tailwind |
| Backend | Node.js + Express + Prisma |
| Base de datos | MariaDB / MySQL |
